# University Timetable Scheduling Using Metaheuristic Adaptive-Elitist Genetic Algorithm
A research on creation of metaheuristic university class timetable scheduling solution.

### Usage:
1. Add instructors
2. Add rooms
3. Add subjects
4. Add sections
5. Generate solution
6. Export or View Result

### Dependencies:
1. Numpy
2. PyQT5
3. psutil
